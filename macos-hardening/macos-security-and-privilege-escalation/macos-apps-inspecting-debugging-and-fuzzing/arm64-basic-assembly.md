* –Ø–∫—â–æ –ø–æ—Ç–æ—á–Ω–∞ —Ñ—É–Ω–∫—Ü—ñ—è –∑–±–∏—Ä–∞—î—Ç—å—Å—è –≤–∏–∫–ª–∏–∫–∞—Ç–∏ –Ω–æ–≤—É —Ñ—É–Ω–∫—Ü—ñ—é —ñ, –æ—Ç–∂–µ, –ø–µ—Ä–µ–∑–∞–ø–∏—Å–∞—Ç–∏ `lr`, –≤–æ–Ω–∞ —Å–ø–æ—á–∞—Ç–∫—É –∑–±–µ—Ä—ñ–≥–∞—î –π–æ–≥–æ –≤ —Å—Ç–µ–∫—É, —Ü–µ –µ–ø—ñ–ª–æ–≥ (`stp x29, x30 , [sp, #-48]; mov x29, sp` -> –ó–±–µ—Ä–µ–≥—Ç–∏ `fp` —Ç–∞ `lr`, —Å—Ç–≤–æ—Ä–∏—Ç–∏ –ø—Ä–æ—Å—Ç—ñ—Ä —ñ –æ—Ç—Ä–∏–º–∞—Ç–∏ –Ω–æ–≤–∏–π `fp`) —ñ –≤—ñ–¥–Ω–æ–≤–ª—é—î –π–æ–≥–æ –≤ –∫—ñ–Ω—Ü—ñ, —Ü–µ –ø—Ä–æ–ª–æ–≥ (`ldp x29, x30, [sp], #48; ret` -> –í—ñ–¥–Ω–æ–≤–∏—Ç–∏ `fp` —Ç–∞ `lr` —ñ –ø–æ–≤–µ—Ä–Ω—É—Ç–∏—Å—è).

* –ó–Ω–∞—á–µ–Ω–Ω—è **`sp`** –∑–∞–≤–∂–¥–∏ –ø–æ–≤–∏–Ω–Ω–æ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏—Å—è –ø—Ä–∏–Ω–∞–π–º–Ω—ñ –Ω–∞ **–∫–≤–∞–¥—Ä–∞—Ç–Ω–æ–º—É** **–≤–∏—Ä—ñ–≤–Ω—é–≤–∞–Ω–Ω—ñ**, —ñ–Ω–∞–∫—à–µ –º–æ–∂–µ –≤–∏–Ω–∏–∫–Ω—É—Ç–∏ –≤–∏–Ω—è—Ç–æ–∫ –≤–∏—Ä—ñ–≤–Ω—é–≤–∞–Ω–Ω—è.

### SIMD —Ç–∞ –ø–ª–∞–≤–∞—é—á—ñ —Ç–æ—á–∫–∏ –†–µ—î—Å—Ç—Ä–∏

–ö—Ä—ñ–º —Ç–æ–≥–æ, —î —â–µ **32 —Ä–µ—î—Å—Ç—Ä–∏ –¥–æ–≤–∂–∏–Ω–æ—é 128 –±—ñ—Ç**, —è–∫—ñ –º–æ–∂—É—Ç—å –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏—Å—è –≤ –æ–ø—Ç–∏–º—ñ–∑–æ–≤–∞–Ω–∏—Ö –æ–ø–µ—Ä–∞—Ü—ñ—è—Ö –∑ –æ–¥–Ω–æ—Ä–∞–∑–æ–≤–∏–º–∏ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è–º–∏ –º–Ω–æ–∂–∏–Ω–Ω–∏—Ö –¥–∞–Ω–∏—Ö (SIMD) —Ç–∞ –¥–ª—è –≤–∏–∫–æ–Ω–∞–Ω–Ω—è –∞—Ä–∏—Ñ–º–µ—Ç–∏–∫–∏ –∑ –ø–ª–∞–≤–∞—é—á–æ—é —Ç–æ—á–∫–æ—é. –í–æ–Ω–∏ –Ω–∞–∑–∏–≤–∞—é—Ç—å—Å—è —Ä–µ—î—Å—Ç—Ä–∞–º–∏ Vn, —Ö–æ—á–∞ –≤–æ–Ω–∏ —Ç–∞–∫–æ–∂ –º–æ–∂—É—Ç—å –ø—Ä–∞—Ü—é–≤–∞—Ç–∏ –≤ **64**-–±—ñ—Ç–Ω–æ–º—É, **32**-–±—ñ—Ç–Ω–æ–º—É, **16**-–±—ñ—Ç–Ω–æ–º—É —Ç–∞ **8**-–±—ñ—Ç–Ω–æ–º—É —Ä–µ–∂–∏–º–∞—Ö, —ñ —Ç–æ–¥—ñ —ó—Ö –Ω–∞–∑–∏–≤–∞—é—Ç—å **`Qn`**, **`Dn`**, **`Sn`**, **`Hn`** —Ç–∞ **`Bn`**.

* –Ø–∫—â–æ **`A`** –¥–æ—Ä—ñ–≤–Ω—é—î 1, —Ü–µ –æ–∑–Ω–∞—á–∞—î, —â–æ **–∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ñ –ø–µ—Ä–µ—Ä–≤–∏** –±—É–¥—É—Ç—å –≤–∏–∫–ª–∏–∫–∞–Ω—ñ. **`I`** –Ω–∞–ª–∞—à—Ç–æ–≤—É—î –≤—ñ–¥–ø–æ–≤—ñ–¥—å –Ω–∞ –∑–æ–≤–Ω—ñ—à–Ω—ñ –∑–∞–ø–∏—Ç–∏ –ø–µ—Ä–µ—Ä–∏–≤–∞–Ω–Ω—è –∞–ø–∞—Ä–∞—Ç–Ω–æ–≥–æ –∑–∞–±–µ–∑–ø–µ—á–µ–Ω–Ω—è (IRQ). –∞ F –ø–æ–≤'—è–∑–∞–Ω–∏–π –∑ **—à–≤–∏–¥–∫–∏–º–∏ –∑–∞–ø–∏—Ç–∞–º–∏ –ø–µ—Ä–µ—Ä–∏–≤–∞–Ω–Ω—è** (FIR).
