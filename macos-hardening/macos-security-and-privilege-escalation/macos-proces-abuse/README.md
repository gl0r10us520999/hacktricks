–†–æ–∑—É–º—ñ–Ω–Ω—è –ø—Ä—ñ–æ—Ä–∏—Ç–µ—Ç—ñ–≤ –ø–æ—Ç–æ–∫—ñ–≤ –ø–µ—Ä–µ–¥–±–∞—á–∞—î –≤–∏–≤—á–µ–Ω–Ω—è —Ç–æ–≥–æ, —è–∫ –æ–ø–µ—Ä–∞—Ü—ñ–π–Ω–∞ —Å–∏—Å—Ç–µ–º–∞ –≤–∏—Ä—ñ—à—É—î, —è–∫—ñ –ø–æ—Ç–æ–∫–∏ –≤–∏–∫–æ–Ω—É–≤–∞—Ç–∏ —ñ –∫–æ–ª–∏. –¶–µ —Ä—ñ—à–µ–Ω–Ω—è –≤–ø–ª–∏–≤–∞—î –Ω–∞ —Ä—ñ–≤–µ–Ω—å –ø—Ä—ñ–æ—Ä–∏—Ç–µ—Ç—É, –ø—Ä–∏–∑–Ω–∞—á–µ–Ω–∏–π –∫–æ–∂–Ω–æ–º—É –ø–æ—Ç–æ–∫—É. –£ macOS —Ç–∞ Unix-–ø–æ–¥—ñ–±–Ω–∏—Ö —Å–∏—Å—Ç–µ–º–∞—Ö —Ü–µ –æ–±—Ä–æ–±–ª—è—î—Ç—å—Å—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –∫–æ–Ω—Ü–µ–ø—Ü—ñ–π, —Ç–∞–∫–∏—Ö —è–∫ `nice`, `renice` —Ç–∞ –∫–ª–∞—Å–∏ —è–∫–æ—Å—Ç—ñ –æ–±—Å–ª—É–≥–æ–≤—É–≤–∞–Ω–Ω—è (QoS).

#### Nice —Ç–∞ Renice

1. **Nice:**
* –ó–Ω–∞—á–µ–Ω–Ω—è `nice` –ø—Ä–æ—Ü–µ—Å—É ‚Äî —Ü–µ —á–∏—Å–ª–æ, —è–∫–µ –≤–ø–ª–∏–≤–∞—î –Ω–∞ –π–æ–≥–æ –ø—Ä—ñ–æ—Ä–∏—Ç–µ—Ç. –ö–æ–∂–µ–Ω –ø—Ä–æ—Ü–µ—Å –º–∞—î –∑–Ω–∞—á–µ–Ω–Ω—è nice –≤ –¥—ñ–∞–ø–∞–∑–æ–Ω—ñ –≤—ñ–¥ -20 (–Ω–∞–π–≤–∏—â–∏–π –ø—Ä—ñ–æ—Ä–∏—Ç–µ—Ç) –¥–æ 19 (–Ω–∞–π–Ω–∏–∂—á–∏–π –ø—Ä—ñ–æ—Ä–∏—Ç–µ—Ç). –ó–Ω–∞—á–µ–Ω–Ω—è nice –∑–∞ –∑–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º, –∫–æ–ª–∏ –ø—Ä–æ—Ü–µ—Å —Å—Ç–≤–æ—Ä—é—î—Ç—å—Å—è, –∑–∞–∑–≤–∏—á–∞–π —Å—Ç–∞–Ω–æ–≤–∏—Ç—å 0.

#### –ö–ª–∞—Å–∏ —è–∫–æ—Å—Ç—ñ –æ–±—Å–ª—É–≥–æ–≤—É–≤–∞–Ω–Ω—è (QoS)

–ö–ª–∞—Å–∏ QoS —î –±—ñ–ª—å—à —Å—É—á–∞—Å–Ω–∏–º –ø—ñ–¥—Ö–æ–¥–æ–º –¥–æ —É–ø—Ä–∞–≤–ª—ñ–Ω–Ω—è –ø—Ä—ñ–æ—Ä–∏—Ç–µ—Ç–∞–º–∏ –ø–æ—Ç–æ–∫—ñ–≤, –æ—Å–æ–±–ª–∏–≤–æ –≤ —Å–∏—Å—Ç–µ–º–∞—Ö, —Ç–∞–∫–∏—Ö —è–∫ macOS, —è–∫—ñ –ø—ñ–¥—Ç—Ä–∏–º—É—é—Ç—å **Grand Central Dispatch (GCD)**. –ö–ª–∞—Å–∏ QoS –¥–æ–∑–≤–æ–ª—è—é—Ç—å —Ä–æ–∑—Ä–æ–±–Ω–∏–∫–∞–º **–∫–∞—Ç–µ–≥–æ—Ä–∏–∑—É–≤–∞—Ç–∏** —Ä–æ–±–æ—Ç—É –Ω–∞ —Ä—ñ–∑–Ω—ñ —Ä—ñ–≤–Ω—ñ –∑–∞–ª–µ–∂–Ω–æ –≤—ñ–¥ —ó—Ö –≤–∞–∂–ª–∏–≤–æ—Å—Ç—ñ –∞–±–æ —Ç–µ—Ä–º—ñ–Ω–æ–≤–æ—Å—Ç—ñ. macOS –∞–≤—Ç–æ–º–∞—Ç–∏—á–Ω–æ —É–ø—Ä–∞–≤–ª—è—î –ø—Ä—ñ–æ—Ä–∏—Ç–µ—Ç–æ–º –ø–æ—Ç–æ–∫—ñ–≤ –Ω–∞ –æ—Å–Ω–æ–≤—ñ —Ü–∏—Ö –∫–ª–∞—Å—ñ–≤ QoS:

1. **–ö–æ—Ä–∏—Å—Ç—É–≤–∞—Ü—å–∫–∏–π —ñ–Ω—Ç–µ—Ä–∞–∫—Ç–∏–≤–Ω–∏–π:**
