## RUBYOPT

Usando esta variable de entorno es posible **agregar nuevos par치metros** a **ruby** cada vez que se ejecuta. Aunque el par치metro **`-e`** no se puede usar para especificar c칩digo ruby a ejecutar, es posible usar los par치metros **`-I`** y **`-r`** para agregar una nueva carpeta a la ruta de carga de bibliotecas y luego **especificar una biblioteca para cargar**.

Crea la biblioteca **`inject.rb`** en **`/tmp`**:

{% code title="inject.rb" %}
```ruby
puts `whoami`
```
{% endcode %}

Crea un script de ruby en cualquier lugar como:

{% code title="hello.rb" %}
```ruby

```
{% endcode %}

Luego, haz que un script ruby arbitrario lo cargue con:
```bash

```

```bash

```
