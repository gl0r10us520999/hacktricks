# 情報収集

特権昇格のミスコンフィグレーションをチェックするだけでなく、現在の状況に関する情報も収集します。

* [x] ユーザー & グループ
* [x] ネットワーク \(インターフェース、arp、ポート、ファイアウォール \(出力多め\)、**ホスト**\)
* [x] プロセス
* [x] スケジュールされたタスク \(出力多め\)
* [x] サービス \(出力多め\)
* [x] インストールされたソフトウェア、プログラムフォルダ
* [x] パッチ
* [x] ドライブ
* [x] 最終更新ファイル

# チェック

* [x] フルコントロール権限のあるファイルとフォルダ
* [x] クオートされていないサービスパス
* [x] 興味深い可能性のあるファイル
* [x] パスワードが含まれるシステムファイル
* [x] 保存されたクレデンシャル
