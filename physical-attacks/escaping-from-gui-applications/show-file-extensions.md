## Windows XP में फ़ाइल एक्सटेंशन कैसे दिखाएं

1. Windows Explorer खोलें। आप यह Start Menu से "My Computer" पर क्लिक करके कर सकते हैं। ध्यान दें कि यह Internet Explorer के समान नहीं है।
2. Explorer में मेनू बार पर "Tools" मेनू पर क्लिक करें (मेनू बार Explorer विंडो के ऊपरी भाग में, विंडो टाइटल के नीचे होता है)। जो पॉपअप मेनू दिखाई देता है उसमें "Folder Options" आइटम पर क्लिक करें।
3. एक डायलॉग बॉक्स प्रकट होता है। विंडो के ऊपरी भाग में "View" टैब पर क्लिक करें।
4. "Hide file extensions for known file types" सेटिंग की तलाश करें और इसे अक्षम करने के लिए इसके बगल में बॉक्स को अनचेक करें।
5. "Apply to All Folders" बटन पर क्लिक करें। फिर "Apply" और फिर "OK" पर क्लिक करें।

## Windows Vista और Windows 7 में फ़ाइल एक्सटेंशन दिखाना

1. Start मेनू पर क्लिक करें। यह गोल आइकन आपकी स्क्रीन के नीचे बाएँ कोने पर होता है।
2. "folder options" टाइप करें (उद्धरण चिह्नों के बिना)। Start मेनू के ऊपरी भाग में दिखाई देने वाली "Folder Options" लाइन पर क्लिक करें।
3. "Folder Options" शीर्षक वाला एक डायलॉग बॉक्स प्रकट होगा। विंडो के ऊपरी भाग में "View" टैब पर क्लिक करें।
4. "Hide extensions for known file types" के लिए बॉक्स को अनचेक करने के लिए क्लिक करें।
5. डायलॉग बॉक्स के नीचे "OK" बटन पर क्लिक करें।

## Windows 8 में फ़ाइल एक्सटेंशन कैसे दिखाएं

1. स्टार्ट स्क्रीन को आमंत्रित करें। ऐसा करने का एक तरीका आपके माउस को आपकी स्क्रीन के नीचे बाएँ कोने पर ले जाना है। Windows 8 में, इससे स्टार्ट स्क्रीन प्रकट होने का कोई संकेत नहीं है, लेकिन यह होगा।
2. "folder options" टाइप करें (उद्धरण चिह्नों के बिना)। मुझे पता है कि स्टार्ट स्क्रीन में कोई खाली फ़ील्ड नहीं है जो सुझाव दे कि आप वास्तव में कुछ टाइप कर सकते हैं। Windows 8 में कई चीजों की तरह, प्रणाली को संचालित करने के लिए गुप्त ज्ञान की आवश्यकता होती है।
3. स्क्रीन के बाएँ तरफ, आपको निराशाजनक संदेश "No apps match your search" मिलेगा। Windows 8 की स्टार्ट स्क्रीन सर्च सुविधा केवल तभी कुछ ढूंढ सकती है जब आप पहले से जानते हों कि इसे कहाँ ढूंढना है और इसे बता सकते हों कि यह कहाँ है।

स्क्रीन के दाहिने तरफ "Settings" लाइन पर क्लिक करें।
4. स्क्रीन के बाएँ तरफ अब "Folder Options" संभावित परिणामों में से एक के रूप में दिखाई देता है। इस पर क्लिक करें।
5. "Folder Options" शीर्षक वाली एक विंडो प्रकट होगी। उस डायलॉग बॉक्स के ऊपरी भाग में "View" टैब पर क्लिक करें।
6. "Hide extensions of known file types" लाइन की तलाश करें और टिक हटाने के लिए इसके बगल में बॉक्स पर क्लिक करें।
7. "OK" बटन पर क्लिक करें।

## Windows 10 में फ़ाइल एक्सटेंशन कैसे दिखाएं

1. स्क्रीन के नीचे टास्क बार पर आइकन पर क्लिक करें ताकि Start मेनू आमंत्रित हो। (आइकन वह है जो Windows लोगो के सफेद संस्करण की तरह दिखता है।)
2. "folder options" टाइप करें (उद्धरण चिह्नों के बिना)। स्क्रीन पर कोई खाली फ़ील्ड नहीं है जो सुझाव दे कि आप कुछ टाइप कर सकते हैं, लेकिन आप कर सकते हैं। बस इसे टाइप करें।
3. "Best match" सूची में दिखाई देने वाले "File Explorer Options" आइटम पर क्लिक करें।
4. "File Explorer Options" शीर्षक वाला एक डायलॉग बॉक्स प्रकट होगा। विंडो के ऊपरी भाग में "View" टैब पर क्लिक करें।
5. "Hide extensions for known file types" आइटम को ढूंढें और टिक हटाने के लिए इसके बगल में बॉक्स पर क्लिक करें।
6. "OK" बटन पर क्लिक करें।

बस इतना ही। अब आपको अपने Explorer विंडोज़ में फ़ाइलों के सही एक्सटेंशन दिखाई देने चाहिए।
