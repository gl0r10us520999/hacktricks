W przypadku oceny phishingu czasami może być przydatne, aby całkowicie **sklonować stronę internetową**.

Zauważ, że możesz również dodać pewne ładunki do sklonowanej strony, takie jak hak BeEF, aby "kontrolować" kartę użytkownika.

Istnieją różne narzędzia, które możesz wykorzystać do tego celu:

## wget
```text
wget -mk -nH
```
## goclone
```bash
#https://github.com/imthaghost/goclone
goclone <url>
```
