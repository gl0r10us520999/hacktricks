Dla oceny phishingu czasami może być przydatne całkowite **sklonowanie strony internetowej**.

Zauważ, że możesz również dodać pewne ładunki do sklonowanej strony, takie jak hak BeEF, aby "kontrolować" kartę użytkownika.

Istnieją różne narzędzia, których można użyć w tym celu:

## wget
```text
wget -mk -nH
```
## sklonujgo
```bash
#https://github.com/imthaghost/goclone
goclone <url>
```
