Bir klavye üzerinden USB ile iletişimi içeren bir pcap dosyanız varsa, aşağıdaki gibi:

![](<../../../.gitbook/assets/image (962).png>)

İletişimde yazılanları almak için [**ctf-usb-keyboard-parser**](https://github.com/TeamRocketIst/ctf-usb-keyboard-parser) aracını kullanabilirsiniz:
```bash
tshark -r ./usb.pcap -Y 'usb.capdata && usb.data_len == 8' -T fields -e usb.capdata | sed 's/../:&/g2' > keystrokes.txt
python3 usbkeyboard.py ./keystrokes.txt
```
Daha fazla bilgi okuyabilir ve bunu nasıl analiz edeceğinize dair bazı betikler bulabilirsiniz:
