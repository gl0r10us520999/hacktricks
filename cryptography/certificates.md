## What is a Certificate

**–ü—É–±–ª—ñ—á–Ω–∏–π –∫–ª—é—á–æ–≤–∏–π —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç** ‚Äî —Ü–µ —Ü–∏—Ñ—Ä–æ–≤–∏–π ID, —è–∫–∏–π –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –≤ –∫—Ä–∏–ø—Ç–æ–≥—Ä–∞—Ñ—ñ—ó –¥–ª—è –ø—ñ–¥—Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è —Ç–æ–≥–æ, —â–æ —Ö—Ç–æ—Å—å –≤–æ–ª–æ–¥—ñ—î –ø—É–±–ª—ñ—á–Ω–∏–º –∫–ª—é—á–µ–º. –í—ñ–Ω –º—ñ—Å—Ç–∏—Ç—å –¥–µ—Ç–∞–ª—ñ –∫–ª—é—á–∞, —ñ–¥–µ–Ω—Ç–∏—á–Ω—ñ—Å—Ç—å –≤–ª–∞—Å–Ω–∏–∫–∞ (—Å—É–±'—î–∫—Ç) —Ç–∞ —Ü–∏—Ñ—Ä–æ–≤–∏–π –ø—ñ–¥–ø–∏—Å –≤—ñ–¥ –¥–æ–≤—ñ—Ä–µ–Ω–æ–≥–æ –æ—Ä–≥–∞–Ω—É (–≤–∏–¥–∞–≤—Ü—è). –Ø–∫—â–æ –ø—Ä–æ–≥—Ä–∞–º–Ω–µ –∑–∞–±–µ–∑–ø–µ—á–µ–Ω–Ω—è –¥–æ–≤—ñ—Ä—è—î –≤–∏–¥–∞–≤—Ü—é, –∞ –ø—ñ–¥–ø–∏—Å –¥—ñ–π—Å–Ω–∏–π, –º–æ–∂–ª–∏–≤–∞ –±–µ–∑–ø–µ—á–Ω–∞ –∫–æ–º—É–Ω—ñ–∫–∞—Ü—ñ—è –∑ –≤–ª–∞—Å–Ω–∏–∫–æ–º –∫–ª—é—á–∞.

–°–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∏ –≤ –æ—Å–Ω–æ–≤–Ω–æ–º—É –≤–∏–¥–∞—é—Ç—å—Å—è [—Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ–π–Ω–∏–º–∏ —Ü–µ–Ω—Ç—Ä–∞–º–∏](https://en.wikipedia.org/wiki/Certificate\_authority) (CA) –≤ –Ω–∞–ª–∞—à—Ç—É–≤–∞–Ω–Ω—ñ [—ñ–Ω—Ñ—Ä–∞—Å—Ç—Ä—É–∫—Ç—É—Ä–∏ –≤—ñ–¥–∫—Ä–∏—Ç–∏—Ö –∫–ª—é—á—ñ–≤](https://en.wikipedia.org/wiki/Public-key\_infrastructure) (PKI). –Ü–Ω—à–∏–π –º–µ—Ç–æ–¥ ‚Äî —Ü–µ [–º–µ—Ä–µ–∂–∞ –¥–æ–≤—ñ—Ä–∏](https://en.wikipedia.org/wiki/Web\_of\_trust), –¥–µ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á—ñ –±–µ–∑–ø–æ—Å–µ—Ä–µ–¥–Ω—å–æ –ø–µ—Ä–µ–≤—ñ—Ä—è—é—Ç—å –∫–ª—é—á—ñ –æ–¥–∏–Ω –æ–¥–Ω–æ–≥–æ. –ó–∞–≥–∞–ª—å–Ω–∏–π —Ñ–æ—Ä–º–∞—Ç –¥–ª—è —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç—ñ–≤ ‚Äî [X.509](https://en.wikipedia.org/wiki/X.509), —è–∫–∏–π –º–æ–∂–µ –±—É—Ç–∏ –∞–¥–∞–ø—Ç–æ–≤–∞–Ω–∏–π –¥–ª—è —Å–ø–µ—Ü–∏—Ñ—ñ—á–Ω–∏—Ö –ø–æ—Ç—Ä–µ–±, —è–∫ –∑–∞–∑–Ω–∞—á–µ–Ω–æ –≤ RFC 5280.

## x509 Common Fields

### **–ó–∞–≥–∞–ª—å–Ω—ñ –ø–æ–ª—è –≤ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞—Ö x509**

–£ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞—Ö x509 –∫—ñ–ª—å–∫–∞ **–ø–æ–ª—ñ–≤** –≤—ñ–¥—ñ–≥—Ä–∞—é—Ç—å –∫—Ä–∏—Ç–∏—á–Ω—É —Ä–æ–ª—å —É –∑–∞–±–µ–∑–ø–µ—á–µ–Ω–Ω—ñ –¥—ñ–π—Å–Ω–æ—Å—Ç—ñ —Ç–∞ –±–µ–∑–ø–µ–∫–∏ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞. –û—Å—å —Ä–æ–∑–ø–æ–¥—ñ–ª —Ü–∏—Ö –ø–æ–ª—ñ–≤:

* **–ù–æ–º–µ—Ä –≤–µ—Ä—Å—ñ—ó** –ø–æ–∑–Ω–∞—á–∞—î –≤–µ—Ä—Å—ñ—é —Ñ–æ—Ä–º–∞—Ç—É x509.
* **–°–µ—Ä—ñ–π–Ω–∏–π –Ω–æ–º–µ—Ä** —É–Ω—ñ–∫–∞–ª—å–Ω–æ —ñ–¥–µ–Ω—Ç–∏—Ñ—ñ–∫—É—î —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç —É —Å–∏—Å—Ç–µ–º—ñ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ–π–Ω–æ–≥–æ —Ü–µ–Ω—Ç—Ä—É (CA), –≥–æ–ª–æ–≤–Ω–∏–º —á–∏–Ω–æ–º –¥–ª—è –≤—ñ–¥—Å—Ç–µ–∂–µ–Ω–Ω—è –∞–Ω—É–ª—é–≤–∞–Ω–Ω—è.
* –ü–æ–ª–µ **–°—É–±'—î–∫—Ç** –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—î –≤–ª–∞—Å–Ω–∏–∫–∞ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞, —è–∫–∏–º –º–æ–∂–µ –±—É—Ç–∏ –º–∞—à–∏–Ω–∞, –æ—Å–æ–±–∞ –∞–±–æ –æ—Ä–≥–∞–Ω—ñ–∑–∞—Ü—ñ—è. –í–æ–Ω–æ –º—ñ—Å—Ç–∏—Ç—å –¥–µ—Ç–∞–ª—å–Ω—É —ñ–¥–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ—é, —Ç–∞–∫—É —è–∫:
* **–ó–∞–≥–∞–ª—å–Ω–∞ –Ω–∞–∑–≤–∞ (CN)**: –¥–æ–º–µ–Ω–∏, –æ—Ö–æ–ø–ª–µ–Ω—ñ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–æ–º.
* **–ö—Ä–∞—ó–Ω–∞ (C)**, **–ú—ñ—Å—Ü–µ–≤—ñ—Å—Ç—å (L)**, **–®—Ç–∞—Ç –∞–±–æ –ø—Ä–æ–≤—ñ–Ω—Ü—ñ—è (ST, S –∞–±–æ P)**, **–û—Ä–≥–∞–Ω—ñ–∑–∞—Ü—ñ—è (O)** —Ç–∞ **–û—Ä–≥–∞–Ω—ñ–∑–∞—Ü—ñ–π–Ω–∞ –æ–¥–∏–Ω–∏—Ü—è (OU)** –Ω–∞–¥–∞—é—Ç—å –≥–µ–æ–≥—Ä–∞—Ñ—ñ—á–Ω—ñ —Ç–∞ –æ—Ä–≥–∞–Ω—ñ–∑–∞—Ü—ñ–π–Ω—ñ –¥–µ—Ç–∞–ª—ñ.
* **–í–∏–¥—ñ–ª–µ–Ω–µ —ñ–º'—è (DN)** –æ—Ö–æ–ø–ª—é—î –ø–æ–≤–Ω—É —ñ–¥–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ—é —Å—É–±'—î–∫—Ç–∞.
* **–í–∏–¥–∞–≤–µ—Ü—å** –≤–∫–∞–∑—É—î, —Ö—Ç–æ –ø–µ—Ä–µ–≤—ñ—Ä–∏–≤ —ñ –ø—ñ–¥–ø–∏—Å–∞–≤ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç, –≤–∫–ª—é—á–∞—é—á–∏ –ø–æ–¥—ñ–±–Ω—ñ –ø—ñ–¥–ø–æ–ª—è, —è–∫ —É –°—É–±'—î–∫—Ç–∞ –¥–ª—è CA.
* **–ü–µ—Ä—ñ–æ–¥ –¥—ñ–π—Å–Ω–æ—Å—Ç—ñ** –ø–æ–∑–Ω–∞—á–∞—î—Ç—å—Å—è —á–∞—Å–æ–≤–∏–º–∏ –º—ñ—Ç–∫–∞–º–∏ **–ù–µ —Ä–∞–Ω—ñ—à–µ** —Ç–∞ **–ù–µ –ø—ñ–∑–Ω—ñ—à–µ**, —â–æ –∑–∞–±–µ–∑–ø–µ—á—É—î, —â–æ–± —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç –Ω–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞–≤—Å—è –¥–æ –∞–±–æ –ø—ñ—Å–ª—è –ø–µ–≤–Ω–æ—ó –¥–∞—Ç–∏.
* –†–æ–∑–¥—ñ–ª **–ü—É–±–ª—ñ—á–Ω–∏–π –∫–ª—é—á**, –∫—Ä–∏—Ç–∏—á–Ω–æ –≤–∞–∂–ª–∏–≤–∏–π –¥–ª—è –±–µ–∑–ø–µ–∫–∏ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞, –≤–∫–∞–∑—É—î –∞–ª–≥–æ—Ä–∏—Ç–º, —Ä–æ–∑–º—ñ—Ä —Ç–∞ —ñ–Ω—à—ñ —Ç–µ—Ö–Ω—ñ—á–Ω—ñ –¥–µ—Ç–∞–ª—ñ –ø—É–±–ª—ñ—á–Ω–æ–≥–æ –∫–ª—é—á–∞.
* **–†–æ–∑—à–∏—Ä–µ–Ω–Ω—è x509v3** –ø–æ–∫—Ä–∞—â—É—é—Ç—å —Ñ—É–Ω–∫—Ü—ñ–æ–Ω–∞–ª—å–Ω—ñ—Å—Ç—å —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞, –≤–∫–∞–∑—É—é—á–∏ **–í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –∫–ª—é—á–∞**, **–†–æ–∑—à–∏—Ä–µ–Ω–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –∫–ª—é—á–∞**, **–ê–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–µ —ñ–º'—è —Å—É–±'—î–∫—Ç–∞** —Ç–∞ —ñ–Ω—à—ñ –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ –¥–ª—è —Ç–æ—á–Ω–æ–≥–æ –Ω–∞–ª–∞—à—Ç—É–≤–∞–Ω–Ω—è –∑–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞.

#### **–í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –∫–ª—é—á–∞ —Ç–∞ —Ä–æ–∑—à–∏—Ä–µ–Ω–Ω—è**

* **–í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –∫–ª—é—á–∞** —ñ–¥–µ–Ω—Ç–∏—Ñ—ñ–∫—É—î –∫—Ä–∏–ø—Ç–æ–≥—Ä–∞—Ñ—ñ—á–Ω—ñ –∑–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è –ø—É–±–ª—ñ—á–Ω–æ–≥–æ –∫–ª—é—á–∞, —Ç–∞–∫—ñ —è–∫ —Ü–∏—Ñ—Ä–æ–≤–∏–π –ø—ñ–¥–ø–∏—Å –∞–±–æ —à–∏—Ñ—Ä—É–≤–∞–Ω–Ω—è –∫–ª—é—á–∞.
* **–†–æ–∑—à–∏—Ä–µ–Ω–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –∫–ª—é—á–∞** —â–µ –±—ñ–ª—å—à–µ –∑–≤—É–∂—É—î –≤–∏–ø–∞–¥–∫–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –¥–ª—è –∞—É—Ç–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ—ó —Å–µ—Ä–≤–µ—Ä–∞ TLS.
* **–ê–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–µ —ñ–º'—è —Å—É–±'—î–∫—Ç–∞** —Ç–∞ **–û—Å–Ω–æ–≤–Ω–µ –æ–±–º–µ–∂–µ–Ω–Ω—è** –≤–∏–∑–Ω–∞—á–∞—é—Ç—å –¥–æ–¥–∞—Ç–∫–æ–≤—ñ —ñ–º–µ–Ω–∞ —Ö–æ—Å—Ç—ñ–≤, –æ—Ö–æ–ø–ª–µ–Ω—ñ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–æ–º, —Ç–∞ —á–∏ —î —Ü–µ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–æ–º CA –∞–±–æ –∫—ñ–Ω—Ü–µ–≤–æ–≥–æ —Å—É–±'—î–∫—Ç–∞ –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω–æ.
* –Ü–¥–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ç–æ—Ä–∏, —Ç–∞–∫—ñ —è–∫ **–Ü–¥–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ç–æ—Ä –∫–ª—é—á–∞ —Å—É–±'—î–∫—Ç–∞** —Ç–∞ **–Ü–¥–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ç–æ—Ä –∫–ª—é—á–∞ –æ—Ä–≥–∞–Ω—É**, –∑–∞–±–µ–∑–ø–µ—á—É—é—Ç—å —É–Ω—ñ–∫–∞–ª—å–Ω—ñ—Å—Ç—å —Ç–∞ –≤—ñ–¥—Å—Ç–µ–∂—É–≤–∞–Ω—ñ—Å—Ç—å –∫–ª—é—á—ñ–≤.
* **–î–æ—Å—Ç—É–ø –¥–æ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó –ø—Ä–æ –æ—Ä–≥–∞–Ω** —Ç–∞ **–¢–æ—á–∫–∏ —Ä–æ–∑–ø–æ–¥—ñ–ª—É CRL** –Ω–∞–¥–∞—é—Ç—å —à–ª—è—Ö–∏ –¥–ª—è –ø–µ—Ä–µ–≤—ñ—Ä–∫–∏ –≤–∏–¥–∞–≤—Ü—è CA —Ç–∞ –ø–µ—Ä–µ–≤—ñ—Ä–∫–∏ —Å—Ç–∞—Ç—É—Å—É –∞–Ω—É–ª—é–≤–∞–Ω–Ω—è —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞.
* **SCT —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç—ñ–≤ CT** –ø—Ä–æ–ø–æ–Ω—É—é—Ç—å –∂—É—Ä–Ω–∞–ª–∏ –ø—Ä–æ–∑–æ—Ä–æ—Å—Ç—ñ, –∫—Ä–∏—Ç–∏—á–Ω–æ –≤–∞–∂–ª–∏–≤—ñ –¥–ª—è –ø—É–±–ª—ñ—á–Ω–æ—ó –¥–æ–≤—ñ—Ä–∏ –¥–æ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞.
```python
# Example of accessing and using x509 certificate fields programmatically:
from cryptography import x509
from cryptography.hazmat.backends import default_backend

# Load an x509 certificate (assuming cert.pem is a certificate file)
with open("cert.pem", "rb") as file:
cert_data = file.read()
certificate = x509.load_pem_x509_certificate(cert_data, default_backend())

# Accessing fields
serial_number = certificate.serial_number
issuer = certificate.issuer
subject = certificate.subject
public_key = certificate.public_key()

print(f"Serial Number: {serial_number}")
print(f"Issuer: {issuer}")
print(f"Subject: {subject}")
print(f"Public Key: {public_key}")
```
### **–†—ñ–∑–Ω–∏—Ü—è –º—ñ–∂ OCSP —Ç–∞ CRL Distribution Points**

CRL –º–æ–∂—É—Ç—å –º—ñ—Å—Ç–∏—Ç–∏ –¥–æ 512 –∑–∞–ø–∏—Å—ñ–≤. –ë—ñ–ª—å—à–µ –¥–µ—Ç–∞–ª–µ–π –¥–æ—Å—Ç—É–ø–Ω–æ [—Ç—É—Ç](https://www.arubanetworks.com/techdocs/ArubaOS%206\_3\_1\_Web\_Help/Content/ArubaFrameStyles/CertRevocation/About\_OCSP\_and\_CRL.htm).

### **–©–æ —Ç–∞–∫–µ –ü—Ä–æ–∑–æ—Ä—ñ—Å—Ç—å –°–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç—ñ–≤**
