# サイドチャネル解析攻撃

サイドチャネル解析攻撃とは、デバイスやエンティティからの情報を、それに間接的な影響を与える他のチャネルやソースから決定することを指します。これは、次の例でよりよく説明できます：

音源に近いガラスシートの振動を分析する。ただし、音源にはアクセスできない。ガラスの振動は音源に影響を受け、監視および分析されれば、音を解読および解釈することができる。

これらの攻撃は、秘密鍵の漏洩やプロセッサ内の操作の発見など、データの漏洩の場合に非常に人気があります。電子回路には、情報が常に漏洩する多くのチャネルがあります。監視および分析は、回路やその内部に関する多くの情報を開示するのに役立つことがあります。
