Na [**Flipper Zero**](https://flipperzero.one/) unaweza:

* **Sikiliza/Pata/Replay masafa ya redio:** [**Sub-GHz**](fz-sub-ghz.md)
* **Soma/Pata/Emulate kadi za NFC:** [**NFC**](fz-nfc.md)
* **Soma/Pata/Emulate lebo za 125kHz:** [**125kHz RFID**](fz-125khz-rfid.md)
* **Soma/Pata/Tuma ishara za Infrared:** [**Infrared**](fz-infrared.md)
* **Soma/Pata/Emulate iButtons:** [**iButton**](../ibutton.md)
* **Tumia kama Bad USB**
* **Tumia kama funguo ya usalama (U2F)**
* **Cheza Snake**
