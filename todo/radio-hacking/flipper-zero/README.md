Z [**Flipper Zero**](https://flipperzero.one/) możesz:

* **Słuchać/Przechwytywać/Odtwarzać częstotliwości radiowe:** [**Sub-GHz**](fz-sub-ghz.md)
* **Czytać/Przechwytywać/Emulować karty NFC:** [**NFC**](fz-nfc.md)
* **Czytać/Przechwytywać/Emulować tagi 125kHz:** [**125kHz RFID**](fz-125khz-rfid.md)
* **Czytać/Przechwytywać/Wysyłać sygnały podczerwieni:** [**Podczerwień**](fz-infrared.md)
* **Czytać/Przechwytywać/Emulować iButtony:** [**iButton**](../ibutton.md)
* **Używać jako Bad USB**
* **Używać jako klucz bezpieczeństwa (U2F)**
* **Grać w Snake**
