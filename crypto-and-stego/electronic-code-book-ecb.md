# ECB

(ECB) –ï–ª–µ–∫—Ç—Ä–æ–Ω–Ω–∞ –∫–æ–¥–æ–≤–∞ –∫–Ω–∏–≥–∞ - —Å–∏–º–µ—Ç—Ä–∏—á–Ω–∞ —Å—Ö–µ–º–∞ —à–∏—Ñ—Ä—É–≤–∞–Ω–Ω—è, —è–∫–∞ **–∑–∞–º—ñ–Ω—é—î –∫–æ–∂–µ–Ω –±–ª–æ–∫ –≤—ñ–¥–∫—Ä–∏—Ç–æ–≥–æ —Ç–µ–∫—Å—Ç—É** –Ω–∞ **–±–ª–æ–∫ —à–∏—Ñ—Ä–æ—Ç–µ–∫—Å—Ç—É**. –¶–µ **–Ω–∞–π–ø—Ä–æ—Å—Ç—ñ—à–∞** —Å—Ö–µ–º–∞ —à–∏—Ñ—Ä—É–≤–∞–Ω–Ω—è. –û—Å–Ω–æ–≤–Ω–∞ —ñ–¥–µ—è –ø–æ–ª—è–≥–∞—î –≤ —Ç–æ–º—É, —â–æ–± **—Ä–æ–∑–¥—ñ–ª–∏—Ç–∏** –≤—ñ–¥–∫—Ä–∏—Ç–∏–π —Ç–µ–∫—Å—Ç –Ω–∞ **–±–ª–æ–∫–∏ –ø–æ N –±—ñ—Ç** (–∑–∞–ª–µ–∂–∏—Ç—å –≤—ñ–¥ —Ä–æ–∑–º—ñ—Ä—É –±–ª–æ–∫—É –≤—Ö—ñ–¥–Ω–∏—Ö –¥–∞–Ω–∏—Ö, –∞–ª–≥–æ—Ä–∏—Ç–º—É —à–∏—Ñ—Ä—É–≤–∞–Ω–Ω—è) —ñ –ø–æ—Ç—ñ–º –∑–∞—à–∏—Ñ—Ä—É–≤–∞—Ç–∏ (—Ä–æ–∑—à–∏—Ñ—Ä—É–≤–∞—Ç–∏) –∫–æ–∂–µ–Ω –±–ª–æ–∫ –≤—ñ–¥–∫—Ä–∏—Ç–æ–≥–æ —Ç–µ–∫—Å—Ç—É, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ —î–¥–∏–Ω–∏–π –∫–ª—é—á.

![](https://upload.wikimedia.org/wikipedia/commons/thumb/e/e6/ECB_decryption.svg/601px-ECB_decryption.svg.png)

–í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è ECB –º–∞—î –∫—ñ–ª—å–∫–∞ –Ω–∞—Å–ª—ñ–¥–∫—ñ–≤ –¥–ª—è –±–µ–∑–ø–µ–∫–∏:

* **–ë–ª–æ–∫–∏ –∑ –∑–∞—à–∏—Ñ—Ä–æ–≤–∞–Ω–æ–≥–æ –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è –º–æ–∂—É—Ç—å –±—É—Ç–∏ –≤–∏–¥–∞–ª–µ–Ω—ñ**
* **–ë–ª–æ–∫–∏ –∑ –∑–∞—à–∏—Ñ—Ä–æ–≤–∞–Ω–æ–≥–æ –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è –º–æ–∂—É—Ç—å –±—É—Ç–∏ –ø–µ—Ä–µ–º—ñ—â–µ–Ω—ñ**

# –í–∏—è–≤–ª–µ–Ω–Ω—è –≤—Ä–∞–∑–ª–∏–≤–æ—Å—Ç—ñ

–£—è–≤—ñ—Ç—å, —â–æ –≤–∏ –∫—ñ–ª—å–∫–∞ —Ä–∞–∑—ñ–≤ –≤—Ö–æ–¥–∏—Ç–µ –≤ –¥–æ–¥–∞—Ç–æ–∫ —ñ **–∑–∞–≤–∂–¥–∏ –æ—Ç—Ä–∏–º—É—î—Ç–µ –æ–¥–∏–Ω —ñ —Ç–æ–π –∂–µ cookie**. –¶–µ —Ç–æ–º—É, —â–æ cookie –¥–æ–¥–∞—Ç–∫–∞ —î **`<username>|<password>`**.\
–ü–æ—Ç—ñ–º –≤–∏ –≥–µ–Ω–µ—Ä—É—î—Ç–µ –Ω–æ–≤–∏—Ö –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á—ñ–≤, –æ–±–∏–¥–≤–∞ –∑ **–æ–¥–Ω–∞–∫–æ–≤–∏–º –¥–æ–≤–≥–∏–º –ø–∞—Ä–æ–ª–µ–º** —ñ **–º–∞–π–∂–µ** **–æ–¥–Ω–∞–∫–æ–≤–∏–º–∏** **—ñ–º–µ–Ω–∞–º–∏ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á—ñ–≤**.\
–í–∏ –≤–∏—è–≤–ª—è—î—Ç–µ, —â–æ **–±–ª–æ–∫–∏ –ø–æ 8B**, –¥–µ **—ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—è –æ–±–æ—Ö –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á—ñ–≤** –æ–¥–Ω–∞–∫–æ–≤–∞, —î **–æ–¥–Ω–∞–∫–æ–≤–∏–º–∏**. –¢–æ–¥—ñ –≤–∏ —É—è–≤–ª—è—î—Ç–µ, —â–æ —Ü–µ –º–æ–∂–µ –±—É—Ç–∏ —á–µ—Ä–µ–∑ —Ç–µ, —â–æ **–≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è ECB**.

–Ø–∫ —É –Ω–∞—Å—Ç—É–ø–Ω–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ. –ó–≤–µ—Ä–Ω—ñ—Ç—å —É–≤–∞–≥—É, —è–∫ —Ü—ñ **2 –¥–µ–∫–æ–¥–æ–≤–∞–Ω—ñ cookie** –º–∞—é—Ç—å –∫—ñ–ª—å–∫–∞ —Ä–∞–∑—ñ–≤ –±–ª–æ–∫ **`\x23U\xE45K\xCB\x21\xC8`**.
```
\x23U\xE45K\xCB\x21\xC8\x23U\xE45K\xCB\x21\xC8\x04\xB6\xE1H\xD1\x1E \xB6\x23U\xE45K\xCB\x21\xC8\x23U\xE45K\xCB\x21\xC8+=\xD4F\xF7\x99\xD9\xA9

\x23U\xE45K\xCB\x21\xC8\x23U\xE45K\xCB\x21\xC8\x04\xB6\xE1H\xD1\x1E \xB6\x23U\xE45K\xCB\x21\xC8\x23U\xE45K\xCB\x21\xC8+=\xD4F\xF7\x99\xD9\xA9
```
–¶–µ –ø–æ–≤'—è–∑–∞–Ω–æ –∑ —Ç–∏–º, —â–æ **—ñ–º'—è –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ —Ç–∞ –ø–∞—Ä–æ–ª—å —Ü–∏—Ö –∫—É–∫—ñ–≤ –º—ñ—Å—Ç–∏–ª–∏ –∫—ñ–ª—å–∫–∞ —Ä–∞–∑—ñ–≤ –ª—ñ—Ç–µ—Ä—É "a"** (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥). **–ë–ª–æ–∫–∏**, —è–∫—ñ —î **—Ä—ñ–∑–Ω–∏–º–∏**, —Ü–µ –±–ª–æ–∫–∏, —è–∫—ñ –º—ñ—Å—Ç–∏–ª–∏ **–ø—Ä–∏–Ω–∞–π–º–Ω—ñ 1 —Ä—ñ–∑–Ω–∏–π —Å–∏–º–≤–æ–ª** (–º–æ–∂–ª–∏–≤–æ, —Ä–æ–∑–¥—ñ–ª—å–Ω–∏–∫ "|" –∞–±–æ —è–∫—É—Å—å –Ω–µ–æ–±—Ö—ñ–¥–Ω—É —Ä—ñ–∑–Ω–∏—Ü—é –≤ —ñ–º–µ–Ω—ñ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞).

–¢–µ–ø–µ—Ä –∑–ª–æ–≤–º–∏—Å–Ω–∏–∫—É –ø—Ä–æ—Å—Ç–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ –≤–∏—è–≤–∏—Ç–∏, —á–∏ —Ñ–æ—Ä–º–∞—Ç —î `<username><delimiter><password>` –∞–±–æ `<password><delimiter><username>`. –î–ª—è —Ü—å–æ–≥–æ –≤—ñ–Ω –º–æ–∂–µ –ø—Ä–æ—Å—Ç–æ **–∑–≥–µ–Ω–µ—Ä—É–≤–∞—Ç–∏ –∫—ñ–ª—å–∫–∞ —ñ–º–µ–Ω –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á—ñ–≤** –∑ **—Å—Ö–æ–∂–∏–º–∏ —Ç–∞ –¥–æ–≤–≥–∏–º–∏ —ñ–º–µ–Ω–∞–º–∏ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á—ñ–≤ —ñ –ø–∞—Ä–æ–ª—è–º–∏, –ø–æ–∫–∏ –Ω–µ –∑–Ω–∞–π–¥–µ —Ñ–æ—Ä–º–∞—Ç —ñ –¥–æ–≤–∂–∏–Ω—É —Ä–æ–∑–¥—ñ–ª—å–Ω–∏–∫–∞:**

| –î–æ–≤–∂–∏–Ω–∞ —ñ–º–µ–Ω—ñ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞: | –î–æ–≤–∂–∏–Ω–∞ –ø–∞—Ä–æ–ª—è: | –î–æ–≤–∂–∏–Ω–∞ —ñ–º–µ–Ω—ñ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ + –ø–∞—Ä–æ–ª—è: | –î–æ–≤–∂–∏–Ω–∞ –∫—É–∫–∞ (–ø—ñ—Å–ª—è –¥–µ–∫–æ–¥—É–≤–∞–Ω–Ω—è): |
| --------------------------- | ---------------- | ----------------------------------- | --------------------------------- |

| 4                           | 4                | 8                                   | 16                                |

# –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤—Ä–∞–∑–ª–∏–≤–æ—Å—Ç—ñ

## –í–∏–¥–∞–ª–µ–Ω–Ω—è —Ü—ñ–ª–∏—Ö –±–ª–æ–∫—ñ–≤

–ó–Ω–∞—é—á–∏ —Ñ–æ—Ä–º–∞—Ç –∫—É–∫–∞ (`<username>|<password>`), —â–æ–± –≤–∏–¥–∞—Ç–∏ —Å–µ–±–µ –∑–∞ —ñ–º'—è –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ `admin`, —Å—Ç–≤–æ—Ä—ñ—Ç—å –Ω–æ–≤–æ–≥–æ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ –∑ —ñ–º'—è–º `aaaaaaaaadmin` —ñ –æ—Ç—Ä–∏–º–∞–π—Ç–µ –∫—É–∫–∏ —Ç–∞ –¥–µ–∫–æ–¥—É–π—Ç–µ —ó—Ö:
```
\x23U\xE45K\xCB\x21\xC8\xE0Vd8oE\x123\aO\x43T\x32\xD5U\xD4
```
–ú–∏ –º–æ–∂–µ–º–æ –ø–æ–±–∞—á–∏—Ç–∏ —à–∞–±–ª–æ–Ω `\x23U\xE45K\xCB\x21\xC8`, —Å—Ç–≤–æ—Ä–µ–Ω–∏–π —Ä–∞–Ω—ñ—à–µ –∑ —ñ–º'—è–º –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞, —è–∫–µ –º—ñ—Å—Ç–∏–ª–æ –ª–∏—à–µ `a`.\
–¢–æ–¥—ñ –≤–∏ –º–æ–∂–µ—Ç–µ –≤–∏–¥–∞–ª–∏—Ç–∏ –ø–µ—Ä—à–∏–π –±–ª–æ–∫ 8B, —ñ –≤–∏ –æ—Ç—Ä–∏–º–∞—î—Ç–µ –¥—ñ–π—Å–Ω–∏–π cookie –¥–ª—è —ñ–º–µ–Ω—ñ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ `admin`:
```
\xE0Vd8oE\x123\aO\x43T\x32\xD5U\xD4
```
## –ü–µ—Ä–µ–º—ñ—â–µ–Ω–Ω—è –±–ª–æ–∫—ñ–≤

–£ –±–∞–≥–∞—Ç—å–æ—Ö –±–∞–∑–∞—Ö –¥–∞–Ω–∏—Ö –æ–¥–Ω–∞–∫–æ–≤–æ —à—É–∫–∞—Ç–∏ `WHERE username='admin';` –∞–±–æ `WHERE username='admin    ';` _(–ó–≤–µ—Ä–Ω—ñ—Ç—å —É–≤–∞–≥—É –Ω–∞ –¥–æ–¥–∞—Ç–∫–æ–≤—ñ –ø—Ä–æ–±—ñ–ª–∏)_

–û—Ç–∂–µ, —â–µ –æ–¥–∏–Ω —Å–ø–æ—Å—ñ–± –≤–∏–¥–∞—Ç–∏ —Å–µ–±–µ –∑–∞ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ `admin` –ø–æ–ª—è–≥–∞—î –≤ —Ç–æ–º—É, —â–æ–±:

* –ó–≥–µ–Ω–µ—Ä—É–≤–∞—Ç–∏ —ñ–º'—è –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞, —è–∫–µ: `len(<username>) + len(<delimiter) % len(block)`. –ó —Ä–æ–∑–º—ñ—Ä–æ–º –±–ª–æ–∫—É `8B` –≤–∏ –º–æ–∂–µ—Ç–µ –∑–≥–µ–Ω–µ—Ä—É–≤–∞—Ç–∏ —ñ–º'—è –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞, —è–∫–µ –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è: `username       `, –∑ —Ä–æ–∑–¥—ñ–ª—å–Ω–∏–∫–æ–º `|` —á–∞—Å—Ç–∏–Ω–∞ `<username><delimiter>` –∑–≥–µ–Ω–µ—Ä—É—î 2 –±–ª–æ–∫–∏ –ø–æ 8B.
* –ü–æ—Ç—ñ–º –∑–≥–µ–Ω–µ—Ä—É–≤–∞—Ç–∏ –ø–∞—Ä–æ–ª—å, —è–∫–∏–π –∑–∞–ø–æ–≤–Ω–∏—Ç—å —Ç–æ—á–Ω—É –∫—ñ–ª—å–∫—ñ—Å—Ç—å –±–ª–æ–∫—ñ–≤, —â–æ –º—ñ—Å—Ç—è—Ç—å —ñ–º'—è –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞, –∑–∞ —è–∫–∏–º –º–∏ —Ö–æ—á–µ–º–æ –≤–∏–¥–∞—Ç–∏ —Å–µ–±–µ, —ñ –ø—Ä–æ–±—ñ–ª–∏, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥: `admin   `

–ö—É–∫–∞ —Ü—å–æ–≥–æ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ –±—É–¥–µ —Å–∫–ª–∞–¥–∞—Ç–∏—Å—è –∑ 3 –±–ª–æ–∫—ñ–≤: –ø–µ—Ä—à—ñ 2 - —Ü–µ –±–ª–æ–∫–∏ —ñ–º–µ–Ω—ñ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ + —Ä–æ–∑–¥—ñ–ª—å–Ω–∏–∫, –∞ —Ç—Ä–µ—Ç—ñ–π - —Ü–µ –ø–∞—Ä–æ–ª—å (—è–∫–∏–π –ø—ñ–¥—Ä–æ–±–ª—è—î —ñ–º'—è –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞): `username       |admin   `

**–¢–æ–¥—ñ –ø—Ä–æ—Å—Ç–æ –∑–∞–º—ñ–Ω—ñ—Ç—å –ø–µ—Ä—à–∏–π –±–ª–æ–∫ –Ω–∞ –æ—Å—Ç–∞–Ω–Ω—ñ–π, —ñ –≤–∏ –±—É–¥–µ—Ç–µ –≤–∏–¥–∞–≤–∞—Ç–∏ —Å–µ–±–µ –∑–∞ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ `admin`: `admin          |username`**
